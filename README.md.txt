GSTify Web - GST Return ProcessorThis is a web-based application for processing E-commerce sales data (from platforms like Amazon, Flipkart, Meesho, etc.) into GST-ready reports.FeaturesMultiple Platform Support: Upload and process sales/return files from various e-commerce sites.B2C & B2B Processing: Handles both Business-to-Consumer and Business-to-Business data.Automated Calculations: Automatically computes Taxable Value, IGST, CGST, and SGST.Comprehensive Reports: Generates key summaries like B2CS, HSN, B2B, and ECO (TCS) reports.Downloadable Formats: Export all generated reports as .xlsx files or a single .zip archive.How to UseVisit the live web page.Set your Business State and ECO GSTINs in the "Initial Setup" section.Upload your sales/return files using the corresponding platform buttons.Navigate to the "View & Download Reports" tab to see the processed data and download your reports.This application runs entirely in your browser. No data is ever uploaded to a server.
